Java如何读取文件

0. 背景

今天在打算运行jstorm中wordcount的例子，需要读取一个文本，这个时候遇到了一个小问题在此记录一下就是如何读取文件的路径。

1. 两种方式

  • class.getResource(String filepath)
  • class.getClassLoader().getResource(String filepath)

    2. demo

    标准的maven项目结构，文本文件放在src/main/resources目录下

简单总结

由于现在使用的普遍是maven项目，
所以当我们使用

  • class.getResource(String filepath)

    filepath 要以/开头

  • class.getClassLoader().getResource(String filepath)

    filepath 不以/开头
